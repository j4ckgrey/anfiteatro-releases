## Releases

This repository tracks the public releases of Anfiteatro.

### Docker Image
The recommended public image is on Docker Hub:
```bash
docker pull j4ckgrey/anfiteatro_releases:latest
```
### Hosted Instance
You can access the latest beta server instance from the hosted instance on Vercel:
```bash
https://anfiteatro-web.vercel.app/
```

### Android Client
Download the `.apk` from the **Releases** section on the right.
